A jQuery search autocomplete plugin with built-in WAI-ARIA

Accessibility for the impaired is often overlooked when building a website. It is a lengthy subject to learn and implement. This plugin is designed to cut down on that effort by providing a search autocomplete (form + list) built with necessary WAI-ARIA HTML. Colors are also chosen for those with color-blindness.
